# üçï Pizza Sales SQL Project

## üìò Overview
This project analyzes pizza sales data using **SQL** to uncover key business insights such as total revenue, top-selling pizzas, and category performance.  
The goal is to demonstrate SQL skills in querying, joining, grouping, and performing analytical operations on real-world data.

---

## üóÇÔ∏è Database Details

| Table Name | Description | Rows |
|-------------|-------------|------|
| `orders` | Contains order details such as order ID, date, and time | 21,350 |
| `order_details` | Contains each order‚Äôs pizza ID and quantity | 48,620 |
| `pizzas` | Stores pizza information like size, type, and price | 90 |
| `pizza_types` | Contains pizza categories, names, and ingredients | 32 |

---

## üßæ Questions Solved
1. Retrieve the total number of orders placed  
2. Calculate total revenue generated from pizza sales  
3. Identify the highest-priced pizza  
4. Find the most common pizza size ordered  
5. List top 5 most ordered pizza types  
6. Find total quantity by pizza category  
7. Analyze order distribution by hour  
8. Category-wise pizza distribution  
9. Average number of pizzas ordered per day  
10. Top 3 pizzas by revenue  
11. Category-wise revenue percentage  
12. Cumulative revenue over time  
13. Top 3 pizzas by revenue per category
